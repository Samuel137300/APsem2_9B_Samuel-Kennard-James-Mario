About the Website

This website was created as part of an Assessment Project (AP) that combines learning from two subjects: Physics and Information and Communication Technology (ICT). The purpose of this project is to integrate scientific knowledge with digital skills by creating an interactive educational website.

From the Physics subject, the materials included in this website focus on topics learned in Grade 9, such as Static Electricity and Magnetism. These topics explain important scientific concepts like electric charge, conductors and insulators, magnetic poles, and magnetic fields. By presenting these materials through slides, videos, and simulations, the website aims to make learning physics more engaging and easier to understand.

From the ICT subject, this project allowed students to apply coding skills to build a real website. The website was developed using HTML, CSS, and JavaScript. HTML is used to create the structure of the web pages, CSS is used to design the layout and visual appearance of the website, and JavaScript is used to add interactive elements that improve the user experience.

Through this collaborative project, students learn how technology can be used to support education. Instead of only reading information from textbooks, students can interact with the material using visual slides, videos, simulations, and quizzes. This approach makes learning more dynamic and helps students better understand scientific concepts.

Overall, this website demonstrates how knowledge from different subjects can work together. Physics provides the scientific concepts, while ICT provides the tools to present those concepts in an interactive digital format. By combining both fields, this project shows how technology can be used to improve learning and make science more accessible and interesting for students.
